## 📃 실행 환경 및 방법

node.js와 npm이 설치되어있어야 합니다.

```
$node --version
v16.15.0
$npm --version
8.13.2
```

프로젝트 clone 후 서버를 실행합니다.

```

npm install
```

기본 주소는 [http://localhost:5173](http://localhost:4000)으로, 서버 실행 후 해당 주소로 접속하시면 됩니다.

### 개발 서버 실행

```bash
npm run dev
    or
npm run start
```

### 프로덕션 빌드 후 실행

```bash
npx run dev
```
